Текстовая игра о постапокалипсисе. В игре нужно вернуться в прошлое и предотвратить ядерную войну. 
Для этого нужно путешествовать по городам, сражаться с врагами и взаимодействовать с другими персонажами игры. 
Файл towns.h содержит основной класс городов. Этот класс наследуется
другими классами конкретных городов в файлах broShelter.h, Megatonna.h, RivetCity.h. 
Файл gameClasses.h содержит основной класс игрока и некотоые наследуемые им классы.
В файле enemy.h содержится класс врагов. 
Файл main.cpp главный исполняющий файл.
